Executable File Types
  • These UNIX-based systems do not rely on file extensions to identify executables.
  • They use a file permission flag that marks a file as executable.
  • Formats include ELF (Executable and Linkable Format) for Linux and Mach-O for macOS.
Script Execution
  • Both systems use shell scripts to automate tasks.
  • These are text files containing UNIX commands.
  • On macOS, the Terminal app provides access to the shell, which is commonly Zsh or Bash.
  • On Linux, the Bash(Bourne-Again Shell) is a widely used default.
  • To be executed, a script file typically needs to be marked as executable using the chmod command.
  • Shell scripts are often identified with a .sh extension
  • Begins with a "shebang" (#!) that specifies the interpreter to be used, such as #!/bin/bash.
